# Simple Portfolio

This is a simple portfolio web page project that showcases various projects and skills. 

## Project Structure

```
simple-portfolio
├── src
│   ├── index.html          # Main HTML document for the portfolio
│   ├── css
│   │   └── styles.css      # Styles for the portfolio web page
│   ├── js
│   │   └── main.js         # JavaScript code for interactivity
│   ├── components
│   │   ├── header.html     # Header component markup
│   │   └── footer.html     # Footer component markup
│   └── data
│       └── projects.json   # JSON data for projects
├── .gitignore              # Files and directories to ignore in version control
├── package.json            # npm configuration file
└── README.md               # Project documentation
```

## Features

- Responsive design
- Dynamic content loading
- Interactive elements

## Setup Instructions

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd simple-portfolio
   ```

3. Install dependencies (if any):
   ```
   npm install
   ```

4. Open `src/index.html` in a web browser to view the portfolio.

## Usage

- Modify the `src/data/projects.json` file to update the projects displayed on the portfolio.
- Customize styles in `src/css/styles.css` to change the appearance of the site.
- Add interactivity by editing `src/js/main.js`.
